# Project Name: StitchFlow

# Purpose:
The Garments Order & Production Tracker System is a web-based platform designed to help small and medium-sized garment factories manage their production workflow. It simplifies the tracking of orders from buyers, manages production stages (cutting, sewing, finishing), monitors inventory, and ensures timely delivery.

# Live Url: https://stitchflow-react.netlify.app/

# Key Features:
Manager Dashboard, Admin Dashboard, User Order Tracking, Payment Gateway and Cash on Delivery.

# NPM Packages Used:
Cors, dotenv, express, firebase-admin, mongodb
